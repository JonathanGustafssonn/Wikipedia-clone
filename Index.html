World Wide Web

[1]The World Wide Web often abbreviated as WWW also known as the Web, is a system for accessing and sharing information through documents and other web resources on the internet. Resources found on the network are available due to web servers and are accessed by programs called web browsers. The specific location of any server or resource on the web are located by using uniform resource locators (URLs).

Documents found on the World Wide Web are often formatted in a markup language called Hypertext Markup Language (HTML). HTML supports a multitude of basic functions such as implementation of text, images, embedded video, and audio files as well as scripts, the language also support hyperlinks which may link to a different document or web page. Information found on the web is transferred back and forth via Hypertext Transfer Protocols (HTTP) there is also an alternative form of protocol called (HTTPS) which is deemed the safer alternative to (HTTP).

Website content is provided to the Web by individual users aswell as governments and companies. The web contains an immense multitude of content be It educational, entertainment, commercial or otherwise, and is considered the worlds most dominant software platform. It is used daily as the primary source of information for millions if not billions of people.

Function

[2]The world wide webs function is to provide a way to access and use different internet resources. This is achieved through linking together different internet resources using URLs and hyperlinks. Resources present in the web are accessed through the use of HTTP or HTTPS protocols which are application-level internet transport protocols.

Navigating between resources found on the web usually starts with either typing in the correct URL for the resource you wish to find into a web browser or through the use of hyperlinks to the corresponding resource. The process of moving between web resources through hyperlinks or the use of a web browser became knonwn as surfing or surfing the web in the 1990s.

When surfing the web using a web browser there a lot of steps the browser goes through in order to link you to the corresponding URL that you desire. These steps consits of the browser taking the namespace of the URL you put in and converting it into an internet protocol address using a globoally distributed system known as Domain Name System (DNS). This returns a an IP adress the browser then requests access to the resource by sending out either a HTTP or HTTPS request over the internet to the computer with the corresponding IP adress. Which then takes you to the resource you want to access. The web browser then analyzes the html and css of the web page and renders it accordingly.

HTML

HyperText Markup Language (HTML) is a standardized markup language used to give web pages and applications their functionality. HTML supports a variety of basic functions such as embedded hyperlinks, video and audio files, simple text, images and scripts. To customise the aesthetics of an html web resource you use a cascading style sheet (CSS). Aforementioned scripts which are supported in a html document are made using javascript, together with html and css these 3 are itegeral to the functionality of the world wide web.

History

[3]Originally designed to enable scientists at Cern to share information with eachother, the world wide web was created in 1989 by British scientist Tim Berners-Lee while he was working for Cern. It took multiple pitches and ultimately a joined effort with Belgian systems engineer Robert Cailliau to get the project greenlit, leading to the development of the project starting in 1990.

In 1991 a version of the world wide web with a more accessible browser referred to as “line-mode” was released to staff at Cern. Early 1993 the National Center for SuperComputing Application released an improved Browser called the Mosaic browser which was more user friendly then previous browsers and also more accessible since it was useable on both pc and mac systems. Following the creation of the Mosaic browser Cern made the source code for the World Wide Web public which resulted in an immense growth of popularity for the world wide web which contributed to the web being able to host approximately 10 000 servers and 10 million users by the end of 1993. The Web is ever improving through the work of the World Wide Web Consortium

Impact on Society

[4][5]The World Wide Web has been one of the main driving force behind the developmment of the information age. Therefore the web has changed our society in multiple ways not all good but not all bad either. Some of the changes the web has brought upon society are stated below.

Positive changes

The web provides features which overall has improved upon society such as.

  • Easy and accessible ways to communicate with others through email, messaging services or videocall services.
  • Easier and faster access to information
  • Online shopping and bank services which provide a safer and more comfortable way to manage money and shopping
  • Plentiful of learning communities which can strive through sharing information among members
  • Online education and learning resources
  • Higher accessibility to global news

Negative changes

The web also has negative effects on society, some of the points below are prominent examples

  • Easier access to illeagal or age inappropriate content
  • health worsening effects due to prolonged screen time
  • Online hackers getting access to private files and stealing data
  • Governments limiting what you can access on the web
